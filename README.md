## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset to understand the factors that influenced passenger survival during the Titanic disaster.

The objective of this project is to analyze passenger information, identify patterns, and extract meaningful insights using data visualization and statistical analysis techniques.

Through data exploration, the project investigates how variables such as passenger class, gender, age, fare, and family relationships influenced survival outcomes.

---

## 🎯 Project Objectives

- Understand the structure of the Titanic dataset
- Perform data cleaning and preprocessing
- Identify missing values and handle them appropriately
- Explore relationships between features
- Visualize survival patterns across different passenger groups
- Generate insights using statistical analysis

---

## 📊 Dataset Description

The Titanic dataset contains information about passengers aboard the RMS Titanic.

| Feature | Description |
|--------|-------------|
| PassengerId | Unique identifier for each passenger |
| Survived | Survival status (0 = No, 1 = Yes) |
| Pclass | Passenger class (1st, 2nd, 3rd) |
| Name | Passenger name |
| Sex | Gender of the passenger |
| Age | Age of the passenger |
| SibSp | Number of siblings/spouses aboard |
| Parch | Number of parents/children aboard |
| Ticket | Ticket number |
| Fare | Ticket fare |
| Cabin | Cabin number |
| Embarked | Port of embarkation |

These attributes help analyze social, demographic, and economic factors that may have affected survival during the disaster.

## 📈 Key Insights

Some important patterns observed during the analysis include:

- Female passengers had a significantly higher survival rate compared to male passengers.
- Passengers traveling in first class had a higher survival probability.
- Younger passengers and children showed higher survival chances.
- Fare and passenger class were strongly related to survival outcomes.

These insights demonstrate how socio-economic status and demographics influenced survival during the Titanic disaster.
